# 🚀 RAG Pipeline (Retrieval-Augmented Generation)

## 📌 Overview
This project implements a complete RAG system that retrieves relevant context from documents and generates accurate responses using LLMs.

## ⚙️ Tech Stack
- Python
- LangChain 
- ChromaDB / FAISS
- Groq / qwen/qwen3-32b


## 🔧 Features
- PDF,TXT , CSV, Excel, Word, JSON files
- Ingestion & chunking
- Embedding generation
- Vector similarity search
- LLM-based answer generation

## 🧠 Key Learnings
- Importance of chunking strategy
- Retrieval quality impacts output
- Handling noisy/unstructured data

```bash
pip install -r requirements.txt
